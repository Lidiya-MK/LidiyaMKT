My Projects

Athonite

An Orthodox Christian app that houses exerpts from books written by Saints. I created this after reading a couple of books from this genre and finding them to be interesting. The app was implemented using flutter.

Book Swap

I always prefer physical books over reading PDFs or other digital formats. But books can be expensive so I thought of "BookSwap" a book exchange platform where users can exchange books they have read for new ones. This was initally meant to be a design project but I am currently implementing it using React and Express.js

Vocation

This project is a job portal for freelancers. It was my final project for my enterprise application development course. I did not work on this project alone. I worked on the freelancers side for the backend and also designed the user interface for the entire system. We used Spring boot for the backend and thymeleaf for templating our HTML pages.

Masinqo

Masinqo is a platform for local artists. Artists can upload their albums and listeners can enjoy them. We created masinqo as a final project for our web and mobile development courses. The mobile application also has a web application implementation. For our backend we used nest js and MongoDB.

EarlyVet

Logistix

This is a shipment tracking and monitoring system It has interfaces for drivers, suppliers, business owners and warehouse staff. This is a design project.

OrganizeMe

This was the second project I created after following a tutotial on Express js. It is a weekly task mangement app. You can mark tasks as important. You can add, edit and delete tasks; you can also set up your profile. Furthermore, the interface has a light and dark mode feature. For the database I used MongoDB.

The platform is meant to facillitate the communication between doctors and cardio vascular patients.

Echo AI

This one is fairly recent. It is a telegram chat bot. To build this I made use of Gemini's API Key. And used python to implement the chat bot. It is fascinating what can be done with a simple python script. I tested the bot out using "Python Anywhere" which allowed me to host my project for free.

Anonymous Quotes

This is a simple Anonymous Quotes webapp. You can add quotes, flag quotes you dont like and like quotes as well. If a quote gets five flags, it gets automatically deleted.

GitHub Data Summary Generator

As part of learning how to write Go code, I worked on a project that allows the user to enter a github user name and in return get a tabulated pdf file that has the name, stars, forks and description of each repository for the github account associated with the user name.

Stroke Prediction

This was one of my projects from my machine learning course. It is a stroke prediction model. But beyond that, the implementation compares five classification algorithms and evaluates their performance using various metrics to check which one is best for the dataset.

CO2 Tracker

This is a web slash linear regression project. It predicts carbon dioxide emission in g/km from car volume and weight. The flask web app allows you to enter volume and weight witin a specified range and makes the predictions using the model I trained.
