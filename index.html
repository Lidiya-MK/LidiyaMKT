My Projects

Athonite

A Christian app that houses exerpts from books written by Orthodox Saints. I created this after reading a couple of books from this genre and finding them to be helpful for my personal life. The app was implemented using flutter.

Vocation

This project is a job portal for freelancers. It was my final project for my enterprise application development course. I worked on it with a partner. We used Spring boot for the backend and thymeleaf for templating our HTML pages.

Book Swap

I always prefer physical books over reading PDFs or other digital formats. But books can be expensive so I thought of "BookSwap" - a book exchange platform where users can exchange books they have read for new ones. This was initally meant to be a design project but I am currently implementing it using React js and Express js

Masinqo

Masinqo is a platform that promotes musicians. The platform has both a mobile and web application implementation. It was my team's final project for our web and mobile development courses. This project also introduced me to Nest js and Typescript.

EarlyVet

Earlyvet is a livestock health monitoring system that makes use of Machine Learning algorithms to make cattle disease predictions. This project is what got me and my team the regional champion title in the seeds for the future program.

Arc

Recently, I have been exploring various design principles. For this particular project, I wanted to follow "Minimalism" to create an elegant landing page. And it turned out to be pretty neat.

Logistix

This is a shipment tracking and monitoring system. It has interfaces for drivers, suppliers, business owners and warehouse staff. Although this was meant to be a hypothetical system for a Requirements Specification project, I went ahead and designed the UI and it turned into a wonderful design project.

OrganizeMe

This was the second project I created after following a tutotial on Express js. It is a weekly task mangement app that allows the user to add, edit and delete tasks or mark them as important. There is also a profile section where the user can update their profile. I used bootstrap for the interface

CardioCare

This was the final project for my "Fundamentals of Software Engineering" course. The platform is meant to streamline information exchange between Cardiologists and their Patients. I remember this was my very first time using Figma. And, whenever I look at the design, I can see that I have come a long way.

Felix

One afternoon, I wanted to experiment with using Blops as image containers for my pages. So I decided to create a landing page for a digital artists community using React js. But I myself don't know why I called it felix :)

Echo AI

Echo is telegram chat bot that is powered by Google's Gemini. I used python to implement it. And tested the bot out using "Python Anywhere" which allowed me to host my project for free.

Anonymous Quotes

This is an Anonymous Quotes webapp. The user can add quotes, like quotes and flag quotes they dislike. If a quote gets five flags, it gets automatically deleted by the system. This project was my first full stack system.

GitHub Data Summary Generator

As part of learning how to program in Go, I worked on a project that allows the user to enter a github user name and in return get a tabulated PDF file that has the name, stars, forks and description of each repository for the github account associated with the user name.

Stroke Prediction

This was one of the projects from my "Machine Learning and Big Data" course. It is a classification model which classifies a patient as likely or not likely to get a stroke. Furthermore, the project compares six well known classification algorithms and evaluates their performance across various evaluation metrics.

CO2 Emission Predictor

Given the volume and weight of a vehicle, this app gives you its estimated CO2 release in g/km. In the background, the flask web app makes use of a linear regression model I pretrained for this purpose.
